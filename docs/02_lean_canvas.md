# リーンキャンパス（真言密教：学び×実践サポート）

## 1. 顧客セグメント（Who）

- 真言密教・空海（弘法大師）の教えに関心がある人
- 学びたいが「続かない」「体系化できない」人
- お寺体験（高野山など）に惹かれるが、日常実践の型がない人

早期採用層（最初の100〜1000人）

- 習慣化アプリの利用経験あり
- 仏教・瞑想・寺社巡りに関心が強い
- “日々の整え”を求めている

## 2. 課題（Problem）

1. 学びが点在していて、理解が続かない
2. 実践がやる気頼みで続かない（型がない）
3. 独学不安：誤解や自己流になりそうで怖い（本来は師の導きが大切）

## 3. 独自価値提案（UVP）

**三密（身・口・意）を毎日回し、学びを“今日の一つの行い”に変える。**  
短い・迷わない・復帰できる。

## 4. 解決策（Solution / MVP）

- 朝：三密チェックイン（3分）
- 昼：学びカード（30秒、行いに変換）
- 夜：整え（懺悔→発願→回向、45秒）
- ログ：週の三密バランス（偏りを見るだけ）

## 5. チャネル（Channels）

- App Store / Google Play（習慣化×仏教×瞑想）
- note / YouTube / Podcast（短い学び発信）
- 高野山・寺院体験・仏教イベント（将来の提携）
- コミュニティ（寺社巡り、瞑想、仏教勉強）

## 6. 収益（Revenue）

Freemium + サブスク（月/年）

- 無料：朝/夜、基本カード、簡易ログ
- 有料：90日コース、阿字観ガイド（音声）、検索/タグ、洞察、監修コンテンツ

将来

- 寺院・講座・体験への送客（提携）
- 監修/公式コンテンツ販売

## 7. コスト（Cost）

- 開発（Expo）
- コンテンツ制作（カード、ガイド）
- 監修（将来）
- サポート/問い合わせ
- マーケ（ASO、SNS）

## 8. 主要指標（Key Metrics）

- D1/D7/D30
- 朝完了率 / 夜完了率
- 途切れ→復帰率（最重要）
- 学びカード完了率
- 無料→有料転換率 / 解約率

## 9. 圧倒的優位性（Unfair Advantage）

- 三密を“日常UX”に落とした設計資産（型）
- コンテンツ運用（短い学び→行動変換）のノウハウ
- 将来：寺院/僧侶監修・協力（正確性と安心）
