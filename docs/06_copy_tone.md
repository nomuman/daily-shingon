# コピー/トーンガイド（アプリ内文言）

## 1. 声（Voice）
- 静か、短い、あたたかい
- “教える”より“一緒に整える”
- 敬意（対象を軽く扱わない）

## 2. 禁止・注意
- 断定：「必ず良くなる」「確実に救われる」
- 脅し：「やらないと不幸になる」
- 競争煽り：「連続◯日を切らすな」
- 上から目線：「あなたは未熟」

## 3. 推奨
- 許す：途切れた前提で「戻れる」
- 小さく：1つだけ、短く
- 具体：今この瞬間にできる行為

## 4. 例文（ボタン/ラベル）
- 朝：整える
- 昼：学ぶ
- 夜：閉じる

完了：
- 「完了」より「整えた」「閉じた」も候補

## 5. 例文（通知）
朝：
- 「3分だけ整えよう（身・口・意）」
夜：
- 「45秒で閉じよう（懺悔→発願→回向）」
復帰：
- 「戻れたら十分。今日は短くでもOK」

## 6. 例文（やさしい言い換え）
×：サボったね
○：今日はここからでいい

×：連続記録が切れた
○：また始められた

×：できなかった原因を分析しよう
○：ひとつだけ整えよう
