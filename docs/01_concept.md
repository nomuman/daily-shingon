# コンセプト（真言密教：学び×実践サポート）

## 1. 目的（北極星）
このアプリの目的は **真言密教の理解を深め、教えを日常で実践し続けるための「補助輪」になること**。

- 「知識を増やす」だけで終わらせない
- 「実践（身・口・意）」が生活の中で回り続けるようにする
- 途切れても戻れる。継続を最優先にする

## 2. コア仮説（価値の中心）
真言密教は「身体・言葉・心」を合わせる実践（身密＝印、口密＝真言、意密＝観想）を重ねることで、仏の身口意と重ねていく、という説明がされることが多い。

→ だからアプリも **三密を“毎日回す”OS**として設計する。

## 3. 提供する体験（毎日の型）
- 朝（3分）：三密チェックイン（身・口・意）
- 昼（30秒）：学びカード（用語→今日の行い）
- 夜（45秒）：整え（懺悔→発願→回向）

※「日記アプリ」にしない。短く、型どおり、毎日同じ。

## 4. スコープ（やる / やらない）
### やる
- 在家が日常で取り組める「入口」と「習慣化」
- 学びを短く提示し、行動に変換する
- 寺院体験・学びの導線（将来）

### やらない（重要）
- 灌頂や口伝の代替
- 効果保証（「必ず人生が良くなる」等の断定）
- 宗教的優劣の断定、他宗派批判、論争

## 5. プロダクト原則（設計の掟）
1. **短い**：朝3分・夜45秒
2. **迷わない**：毎日同じ流れ、選択肢は少なく
3. **復帰が偉い**：連続より「戻る」
4. **敬意**：宗教的対象への敬意、軽薄さを避ける
5. **正確さ優先**：出典方針と監修導線（後で入れる）
6. **プライバシー最優先**：ログは基本端末内

## 6. 成果の扱い（表現方針）
- 「人生が良くなる」は断言しない
- 代わりに「整う確率が上がる」「気づきが増える」「丁寧さが増える」など、本人の体験に寄せる
- 体験の個人差を前提にする

## 7. 用語（アプリ内の最小定義）
- 身：姿勢・所作・呼吸（身体の整え）
- 口：真言／誓い／発する言葉（言葉の整え）
- 意：観想／注意／気づき（心の整え）
- 懺悔：責めるためではなく、整えるための省察
- 発願：明日一つだけ整える宣言
- 回向：功徳を自他へ回し、ともに歩む
