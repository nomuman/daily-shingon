このアプリについて

まいにち真言は、真言密教の学びを日常に落とし込むための小さな支えです。迷わず、戻れることを大切にします。

毎日の流れ

朝｜整える
約3分。身・口・意を整える。
昼｜学ぶ
約30秒。短い学びから行いを一つ。
夜｜閉じる
約45秒。懺悔 → 発願 → 回向。

三密（身・口・意）

三つの軸が常に見えるように設計しています。

大切にしていること

  • 短く、迷わない導線
  • 途切れても戻れる体験
  • 断定せず、敬意を持った言葉
  • 個人ログは端末内に保存
