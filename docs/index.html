このアプリについて

まいにち真言は、真言密教の学びを日常に落とし込むための「入口」をつくるアプリです。宗教の代替ではなく、短く、迷わず、 戻れることを大切にします。

毎日の流れ

朝3分・昼30秒・夜45秒。ひとつだけ整える。

朝｜整える

約3分

身・口・意を整え、今日の行いを一つ選びます。

昼｜学ぶ

約30秒

短い学びカードで、今日の行いを具体化します。

夜｜閉じる

約45秒

懺悔 → 発願 → 回向で一日を静かに締めます。

三密（身・口・意）を中心に

三つの軸が常に見えるように設計しています。

身｜姿勢・所作
口｜言葉・真言
意｜観想・心

大切にしていること

  • 短く、迷わない導線
  • 途切れても戻れる体験
  • 断定せず、敬意を持った言葉
  • 個人ログは端末内に保存
