# コンテンツ方針（正確さ・敬意・安全）

## 1. 大原則
- このアプリは「宗教の代替」ではなく「日常実践のサポート」
- 断定しない（効果保証しない）
- 他宗派・他者への批判をしない
- 専門領域（灌頂・口伝の内実）には踏み込まない

## 2. 取り扱う範囲（OK）
- 三密（身/口/意）の入門的説明と、日常への落とし込み
- 在家が取り組みやすい短い実践（姿勢、呼吸、短い誓い、短い観想）
- 阿字観の「入口」（初心者向け、短時間、体験への導線）
- 日常五心のような生活倫理としての説明（反省・感謝など）

## 3. 取り扱わない範囲（NG / 注意）
- 修法・伝法・灌頂などの「具体手順」を詳細に説明すること
- 「これを唱えれば必ず◯◯」のような効能の断定
- 医療・治療・精神疾患の改善を示唆する表現
- 個別の霊的判断・鑑定・占いへの誘導

## 4. 出典・監修方針
- “基礎概念の説明”は一次情報（寺院・学術・百科）を優先
- 文章は短く、誤解を招きやすい箇所は注釈を入れる
- 将来：寺院/僧侶監修の導線（監修者名・範囲の明示）

## 5. チェックリスト（公開前）
- [ ] 効果保証っぽい断定がない
- [ ] 他者や宗派の批判がない
- [ ] 専門領域の手順を過剰に書いていない
- [ ] “入口”として短く、行動に変換できる
- [ ] 読後に「自責」を増やす文言になっていない
- [ ] 出典が追える
